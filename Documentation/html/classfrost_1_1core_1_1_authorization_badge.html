Detailed Description

template<typename T>
class frost::core::AuthorizationBadge< T >

This file contains the AuthorizationBadge class, which is used to grant access to private constructors.

You can add an authorization badge as a parameter to a method by specifying which class is the target of the badge, so that this method can only be called by the class specified in the badge template.
