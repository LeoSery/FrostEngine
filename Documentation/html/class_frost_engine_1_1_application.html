Browse all GameObjects in the current scene.

Go through the Tree of GameObjects of the currently active scene, add each object to the rendering queue, call its 'Update()' method and recursively do the same with each of its children.

◆ PhysUpdate()

void FrostEngine::Application::PhysUpdate ( )
private

Computes and applies collisions for each GameObject in the rendering list.

Performs the physical rendering of the engine. For each GameObject in the renderer, retrieves its BoxCollider component, and checks whether it collides with all other objects in the scene.

If it does, this object is added to the 'Hits' map, with a 'CollisionData' vector for each object with which it collides. Then, for each Hit of each GameObject, we call its 'OnCollisionEnter()' method, passing it 'CollisionData' as a parameter.

Finally, we move the object with the corresponding collision data.

◆ Run()

void FrostEngine::Application::Run ( )

Engine main loop method.

Infinite engine loop that runs as long as the engine is open, updating the engine state at each frame. It renders in the following order: Update user > Update GameObjects > Update Components > Update physics > Update inputs > Update render > Update logger.
